# Web Epica XD

Proyecto de página web para publicar mensajes y subir archivos.

## Características

- Publicar mensajes con nombre de usuario
- Subir múltiples archivos: imágenes, PDFs, documentos, hojas de cálculo
- Vista previa de imágenes antes de publicar
- Descargar archivos publicados
- Sistema de votos para cada publicación
- Estilo moderno con diseño responsive

## Archivos

- `index.html`: estructura de la página
- `styles.css`: estilos y diseño
- `script.js`: lógica principal de la app
- `helpers.js`: utilidades de carga y formato de archivos

## Uso

1. Abre `index.html` en el navegador.
2. Ingresa tu nombre y publica mensajes con archivos.
3. Recarga la página para ver las publicaciones guardadas localmente.
